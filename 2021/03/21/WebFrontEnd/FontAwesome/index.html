在网站中使用 Font Awesome 图标

2021-03-21 前端 168 字
Font Awesome Wed前端 icon

Font Awesome 是一套绝佳的图标字体库和CSS框架，Font Awesome 字体为您提供可缩放矢量图标,它可以被定制大小、颜色、阴影以及任何可以用CSS的样式

文章内容根据菜鸟教程 Font Awesome 图标 的思路来写，但原教程使用的是 V4.7.0 版本，而实际上 Font Awesome 已经更新到 V5 版本，原 V4.7.0 版本已不再被支持，两个版本有较大的区别，本文就 V5 版本的使用进行讲解
